Reports deprecated commands in GitHub Actions workflows.

Example:


  run: |
    # Deprecated commands
    echo "::set-output name=foo::bar"
    echo "::save-state name=foo::bar"
    echo "::set-env name=foo::bar"
    echo "::add-path::$HOME/.local/bin"

  • Deprecating save-state and set-output commands
  • Deprecating set-env and add-path commands
